# TODO Memo

TODO를 입력해서 목록에 추가하고, 삭제할 수 있는 간단한 앱입니다.

## 기술 스택

- React 18
- TypeScript
- Vite
- Vitest (Jest 호환 API) + React Testing Library

## 실행 방법

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run test     # 테스트 감시 모드
npm run test:run # 테스트 1회 실행
npm run build    # 프로덕션 빌드
```

## 테스트 체크리스트

- "a" 추가됨?
- Enter로 추가됨?
- 빈 값 추가 안 됨?
- 삭제 잘 됨?
- 연속 추가해도 OK?
